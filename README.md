## About me

Full-stack developer who likes working close to the data. I build products that live on top of real databases — ERPs, public-spending datasets, solar telemetry, LLMs that query databases in natural language. I care about the whole pipeline: the schema, the ingestion, the query, and the UI that makes it make sense.

Currently at **Accenture** as a Business Architecture Associate — SQL troubleshooting on global enterprise databases, incident response, Azure deployments, and AI-assisted debugging workflows.
